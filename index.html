Welcome to Shape To Music

Metamaterials are materials with “meta”, or extraordinary properties. Materials can be given these “meta” properties through optimization of their structure, or topology. Many topologies can be generated for the same material; while it can be difficult to see topological differences, it can be easier to hear them. This is done by creating songs for each spinodoid (topological image). The more similar two topologies are, the more similar their songs will be (and vice versa). This allows trends between properties and topologies to be determined.

Two approaches have been used to generate songs: the FEA (Finite Element Analysis) Approach and the Image-Based Approach. A description of each approach, as well as the music generated by each, can be found by clicking the corresponding button.
